# Project 1 — Cybersecurity Risk Register

**Organization:** White Plains Hospital (simulated acute care hospital)
**Prepared By:** Nadhir Choudhury — GRC Analyst
**Framework:** NIST CSF | HIPAA Security Rule
**Tool:** Microsoft Excel
**Date:** April 2026

---

## What This Project Is

A complete cybersecurity risk register built for a
simulated acute care hospital environment.

Identifies 15 cybersecurity risks across four categories
— people, technology, processes, and third-party vendors.
Each risk is scored using likelihood and impact methodology
with treatment strategies, risk owners, and target dates.

---

## What Is Inside the File

- Cover page with document information
- Risk register with 15 fully documented risks
- Color coded risk heat map
- Executive summary with top priority findings
- Prioritized remediation recommendations

---

## Skills Demonstrated

- Risk identification across multiple categories
- Likelihood x Impact scoring methodology
- Risk treatment strategies — Avoid, Reduce, Transfer, Accept
- Risk register documentation and maintenance
- Alignment to NIST CSF five core functions
- Alignment to HIPAA Security Rule requirements
- Executive risk reporting and heat map visualization
